# 上海海洋大学urp自动评教程序
摆脱手动评教的苦海
## 依赖的库
* bs4
* requests
* pillow
## 原理
* 模拟登陆获取cookie
* 获取评价列表
* 模拟POST评价信息

## 使用教程
* Python版本：3.0及以上
* 运行main.py
